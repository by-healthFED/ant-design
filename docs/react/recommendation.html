Third-Party Libraries

antd is designed to provide high-quality React UI components which following Ant Design Specification. So, we are not going to implement other functions which are not relative to antd's target, and recommend to use excellent third-party library which came from React community:

Routerreact-router
Layoutreact-blocks
Drag and dropreact-dnd react-draggable
Code Editorreact-codemirror
Rich Text Editorreact-quill react-draft-wysiwyg react-lz-editor(Ant Design style)
Color Pickerrc-color-picker react-color
Media Queryreact-responsive react-media
Copy to clipboardreact-copy-to-clipboard
Document head managerreact-helmet react-document-title
Iconsreact-fa react-icons
